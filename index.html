Dotsimus - Basic

To get it going, the only thing that you need to do is to add Dotsimus to your server, enjoy!

Dotsimus - Premium

Dotsimus Premium is an ongoing subscription service, to enroll into you need to do the following things.

  1. Add Dotsimus to your server.
  2. Pay for Premium over at GitHub
  3. Message Dotsimus bot your GitHub username along with server id or server invite, shortly after messaging premium functionality will be activated for your community.
